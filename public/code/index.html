Hugo

Is a great static site generator and what I use to run this site. It’s based on the Go language, which is yet another computer programming language. How many do we need? Golang has some awesome features, and to get under the skin of Hugo, you probably need to get a grasp of Hugo’s idiosyncratic mode of programming
